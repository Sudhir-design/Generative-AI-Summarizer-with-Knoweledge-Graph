#  Generative AI Summarizer with Knowledge Graph

This project is a **Streamlit web application** that uses **Hugging Face Transformers (BART)** to summarize long text and **SpaCy + NetworkX** to extract named entities and build an interactive knowledge graph.

---

##  Features
-  **Summarization** ‚Üí Generates concise summaries of long text using Hugging Face `facebook/bart-large-cnn`  
-  **Knowledge Graph** ‚Üí Extracts people, organizations, and locations with SpaCy (NER) and builds an interactive graph using PyVis  
-  **Streamlit UI** ‚Üí Clean web interface to paste text and visualize results  
-  **Entity Filtering** ‚Üí Keeps only meaningful entities (PERSON, ORG, GPE) for clarity  
-  **Future Work** ‚Üí Add automatic ROUGE evaluation for summaries
